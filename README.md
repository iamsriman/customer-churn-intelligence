Customer Churn Intelligence Dashboard



This project analyzes telecom customer data to identify churn patterns and high-risk customers.

Dataset
IBM Telco Customer Churn Dataset (~7,000 customers)

Tools Used
Python
SQL
Power BI

Dashboard Insights
• Month-to-month contracts have the highest churn
• Customers with tenure less than 12 months churn more
• Electronic check payment method shows higher churn risk
• High-value customers contribute significant revenue loss

Outcome
Built an interactive dashboard enabling companies to monitor churn trends and improve customer retention.
